這是一個使用 [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app) 建立的 [Next.js](https://nextjs.org) 專案。

## 開始使用

首先，執行開發伺服器：

```bash
npm run dev
# 或
yarn dev
# 或
pnpm dev
# 或
bun dev
```

接著在瀏覽器中開啟 [http://localhost:3000](http://localhost:3000) 即可查看結果。

您可以透過修改 `app/page.js` 來開始編輯頁面。當您編輯檔案時，頁面會自動更新。

本專案使用 [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) 自動最佳化並載入 [Geist](https://vercel.com/font) 字體系列。
